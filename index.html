A Dynamic J2EE Java Servlet & Python OpenCV Neural Network Web Application

GitHub Pages Notice
You are seeing this page because this repository is deployed on GitHub Pages, which only supports static file hosting. Because this application requires a Java (Tomcat) application server, a MySQL database, and Python AI weights to execute, the active detector cannot run directly on this static hosting site.
View Deployment Guide

Option A: Deploy to Render (Cloud)

  • 1
    Create a free MySQL Database on database cloud providers like Aiven.io or Clever Cloud. Copy your connection URL, username, and password.
  • 2
    Sign up at Render.com, create a new Web Service, and link your GitHub repository amritanshu2711/AI-Object_Detection.
  • 3
    Select Docker as the Runtime environment (Render will read our preconfigured Dockerfile automatically).
  • 4
    In the Render service environment settings, add these variables:
    DB_URL = [Your MySQL JDBC URL]
    DB_USER = [Your Username]
    DB_PASSWORD = [Your Password]
    PYTHON_CMD = /opt/venv/bin/python
  • 5
    Click Deploy Web Service. Render will build and host your J2EE backend with Python OpenCV!

Option B: Run Locally (Recommended)

  • 1
    Install Docker Desktop on your local computer.
  • 2
    Clone your repository to your local drive:
    git clone https://github.com/amritanshu2711/AI-Object_Detection.git
  • 3
    Open a terminal in the cloned directory and run Docker Compose:
    docker-compose up --build
  • 4
    This will download the AI models, compile the J2EE application, start MySQL, and initialize the database automatically.
  • 5
    Open your browser and navigate to:
    http://localhost:8080/ai_object_detection_system
